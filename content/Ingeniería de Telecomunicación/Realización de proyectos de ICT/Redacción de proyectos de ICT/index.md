Este último tema fundamentalmente se dedica al estudio del contenido de un proyecto técnico de ICT. El tema se divide en dos partes. Una primera parte donde se estudia el proyecto desde un punto de vista teórico; esto es partes que debe contener; el proceso de consulta e información, la ejecución del mismo,… Una segunda parte donde se explica la forma de hacer la memoria del proyecto de ICT, y además se facilita un ejemplo de memoria.
